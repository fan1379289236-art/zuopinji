## 00:11 用 ffmpeg 压缩首屏视频
- `ffmpeg-static` 的 npm 安装因 GitHub 二进制 IP 被墙失败（ETIMEDOUT）；本机也无 ffmpeg。
- 改用 Python 包 `imageio-ffmpeg`（ffmpeg 二进制备在 PyPI wheel 内，不需 GitHub）：在 `python/envs/video` venv 中 `pip install imageio-ffmpeg`，ffmpeg 路径通过 `imageio_ffmpeg.get_ffmpeg_exe()` 取得（v7.1，位于 `envs/video/Lib/site-packages/imageio_ffmpeg/binaries/`）。
- 压缩参数：`-vf scale=1280:-2 -c:v libx264 -crf 30 -preset medium -an -pix_fmt yuv420p -movflags +faststart`。
- 结果：`hero.mp4` 9.6MB→3.4MB（2560x1440→1280x720，去音轨）；`hero2.mp4` 63MB→6.4MB。已替换原文件并清理临时文件。
