- 安装 `gsap` + `@gsap/react`（`npm install`，registry 不受 GitHub binary 限制影响）。
- 新建 `src/lib/gsap.js`：注册 ScrollTrigger/useGSAP、全局默认 `ease: expo.out`、reduced-motion 检测、ScrollTrigger.config(ignoreMobileResize)。

- Navbar：useGSAP 下落淡入开场。
- App：字体/load 后 `ScrollTrigger.refresh()` 防止大标题换字体导致触发位置偏移。
